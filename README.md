# SpectrumCov软件介绍

## :wrench:功能

- 自动化的计算中子或伽马能量注量转换成周围剂量当量、个人剂量当量和照射量

## :house:特点

- 无需编程基础

## :de:软件依赖

- root 6.22
- Qt5.15

## :fire:软件安装

- linux

完整版安装

```bash
. install.sh
```

运行

```bash
./SpectrumCov
```

点击帮助查看使用说明
